1. Run extract_words.py in the terminal. Make sure that you have Dracula.txt
or any other novel of your choosing in .txt format. Once you run the code, you
should see allwords.txt, uniquewords.txt, and wordfrequency.txt.
2. unique_text.txt will be in a3_novelvisualization directory
3. open a3_novelvisualization directory and press PLAY to generate a word cloud
4. wordfrequency.txt will be in a3_wordfrequency directory
5. open a3_wordfrequency directory and press PLAY for a visualization of the
relationship between word frequency and number of words with those of
frequencies
* disregard test.pde/test_text as they were used for testing our programs
