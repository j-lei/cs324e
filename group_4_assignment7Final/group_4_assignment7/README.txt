How to run the game: open file group_4_assignment7.pde and select run. You use w, a, s, d keys to move the player around (w for up, a for left, s for down, and d for right) In addition, press p for pausing the game, m for restarting the game, r for resuming the game, and q for quitting the game. Also included in the zip file is a GIF of a run of the gameplay to fulfill the Animation Artifact extra credit.
